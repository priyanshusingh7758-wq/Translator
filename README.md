# 🌍 Translator Web App

## 📌 Description
The Translator Web App uses Microsoft Azure API to convert text into multiple languages. It features a dashboard for API key and region input, supports 10+ languages, and includes Bhojpuri via Hindi. The app offers a colorful UI, fast translation, and user-friendly design.

---

## 🚀 Features
- 🌐 Multi-language translation (10+ languages)
- 🗣️ Bhojpuri support (via Hindi)
- 🎛️ Dashboard UI (Translator + Settings)
- 🔐 API Key & Region input
- ⚡ Fast and real-time translation
- 🎨 Colorful and modern UI/UX

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Microsoft Azure Translator API

---

## 📂 Project Structure
translator/

---

## ⚙️ How to Run

1. Open project in VS Code  
2. Run `index.html` in browser  
3. Enter:
   - Azure API Key  
   - Region  
4. Start translating 🎉  

---

## ⚠️ Note
- Bhojpuri is not officially supported → uses Hindi as fallback
